# 🪙 COIN SUIKA GAME

コインをテーマにしたスイカゲーム！  
同じコインを合体させて、より大きなコインを目指そう！

## 🎮 遊び方

- **マウス移動 / タップ** → コインの落とし位置を決める
- **クリック / タップ** → コインを落とす
- 同じコインが当たると **合体** して次のレベルへ！
- 連続合体で **コンボボーナス** 発動！
- 最大コイン「**MIRCOコイン**」を作ろう！

## ✨ 特徴

- 🌟 宇宙背景 + 流れる星
- 💥 マージエフェクト（パーティクル・衝撃波）
- 📳 スクリーンシェイク
- 🔊 Web Audio APIサウンド
- 🔥 コンボシステム（最大×8倍）
- 👑 LEGEND演出（最大コイン出現時）
- 🏆 ハイスコア自動保存
